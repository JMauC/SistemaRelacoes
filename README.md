# SistemaRelacoes

##Funcionamento
Esse programa recebe um ou mais arquivos ".txt" que sigam o padrão:
```
inteiro (número de elementos do domínio)
inteiro (número de pares ordenados da relação)
inteiro1 inteiro2 (inteiro 1 é a coordenada x do par ordenado, inteiro 2 a coordenada y)
.
.
.
inteiro inteiro (ultimo par ordenado)
```

Onde :
1. Os inteiros são algarismos, não escritos por extenso.
2. Os parenteses e seu conteúdo não existem.
###Exemplo: 
```
6
5
0 1
0 0
1 5
2 2
4 4
```
